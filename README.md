jawamp
======

**jawamp** is Just Another JAva implementation of [WAMP](http://wamp.ws/ "WebSocket Application Messaging Protocol").

Quick start
-----------

```xml
<dependency>
    <groupId>com.github.ghetolay</groupId>
    <artifactId>jwamp</artifactId>
    <version>0.0.1-SNAPSHOT</version>
</dependency>
```
